Lesson 1: Set Up A Scene

Before we can start filming we’re going to need a set, so let’s walk into the room and set up a few things to get started.

The first thing we'll need to do is define a renderer. We've provided that in the editor for you, ie: var renderer = new THREE.WebGLRenderer();

// set the scene size var renderer = new THREE.WebGLRenderer(); var camera = new THREE.PerspectiveCamera( VIEW_ANGLE, ASPECT, NEAR, FAR); var scene = new THREE.Scene(); scene.add(camera); camera.position.z = 300; renderer.setSize(WIDTH, HEIGHT); $container.append(renderer.domElement);
